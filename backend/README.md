# SmartQueue India Backend (Flask)

This is the Python backend for the AI Appointment Management System.

## Setup Instructions

1. **Create a Virtual Environment:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Server:**
   ```bash
   python app.py
   ```

The server will run at `http://127.0.0.1:5000`.
